Requesting a grant

An agent has two verbs. POST an Operation to /api/operations to run one action just-in-time — it executes if already granted, otherwise you get an approval URL and a re-run succeeds. Or POST a GrantRequest (a list of capabilities) to /api/grant-requests to pre-approve a scoped bundle for later (or granular request -f req.json).
